📊 Financial Risk Profiling System
🚀 Project Overview

This project is an end-to-end Financial Risk Profiling System that predicts borrower default risk using machine learning and provides risk-based loan decisions and financial recommendations.

📂 Dataset

Lending Club – Accepted Loans Dataset (~300K records)

⚙️ Features

Loan Amount

Loan Term

Annual Income

Debt-to-Income Ratio (DTI)

FICO Score

Employment Length

Home Ownership

Loan Purpose

🧠 Model

Random Forest Classifier

ROC-AUC ≈ 0.71

📈 Output

Risk Score (0–100)

Risk Category (Low / Medium / High)

Loan Decision (Approved / Conditional / Rejected)

Explainable Risk Factors

Product Recommendations

🌐 Deployment

Built using Streamlit
